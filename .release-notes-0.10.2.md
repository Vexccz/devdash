# DevDash v0.10.2

Git panel now lives in the project detail overview, not just the Projects grid.

## What's new

- **Git card on Overview tab** — branch, ahead/behind counts, dirty-file breakdown (modified / staged / untracked), and the latest commit (hash, message, author, date) right inside each project's detail view.
- **Commit / Pull / Diff / Open repo buttons** wired in-place. The Commit button opens QuickCommitModal (stage-all + push toggle); Diff opens the existing diff viewer; Pull runs the safe pull flow with an inline result toast.
- **Refresh with remote fetch** so ahead/behind stays current without leaving the screen.
- Buttons disable themselves when there's nothing to do (e.g. Diff greyed out on a clean tree, Open repo greyed when `githubUrl` is missing).

## Why

Zero-context commits: you're already looking at a project's metrics, uptime, and bundle size — now you can commit and push without bouncing back to the Projects grid.
