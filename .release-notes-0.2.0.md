# DevDash v0.2.0

Thirteen new features on top of the v0.1.0 foundation. Projects, Deploys, and Settings are preserved; three new tabs (Uptime, Time, Deps) sit alongside them, and every project card opens a detail modal with its own Logs / Env / Time / Deps / Heatmap / Screenshots / Release tabs.

## Highlights

- **Log tail viewer** - live stdout/stderr of managed dev servers with ERROR/WARN color coding, keyword filter, auto-scroll, kill button.
- **Smart dev server launcher** - detects Vite, Next, Expo, Flutter, FastAPI/Uvicorn, generic Node; managed child process with live port + status dot.
- **Endpoint health checker** - scheduled HTTP GET, latency history, per-project sparklines on the Uptime tab, toast on up->down.
- **Env var manager** - scans .env/.env.local/.env.production, compares to .env.example, inline editor with masked values, clone-from-project dropdown.
- **Command palette** (Ctrl+K / Cmd+K) - fuzzy search every project and action.
- **Task timer per project** - auto-starts on detail open, auto-stops on idle; weekly bar chart on the Time tab.
- **Changelog generator** - groups commits by Conventional Commit type, suggests bump kind, writes CHANGELOG.md.
- **One-click release** - bumps package.json, commits, tags, pushes, calls `gh release create`, uploads dist/*.exe and dist/*.apk.
- **Error budget tracker** - local log scanning by default, optional Sentry via auth token in Settings.
- **Bundle size watch** - mtime-driven, records dist/ size, warns on >10% growth vs 7-day avg.
- **Dependency outdated checker** - weekly `npm outdated --json`, major/minor/patch counts per project.
- **GitHub activity heatmap** - 90-day commit calendar with current + longest streaks.
- **Screenshot gallery** - daily captures via headless Electron BrowserWindow, thumbnails + timelapse.

## Install

Download `DevDash-Setup-0.2.0.exe`, run, pick install location. Existing v0.1.0 users keep their config and cache.db (migrated in place). Add tokens and project Sentry DSNs in Settings.

- `npm run dist` -> `dist/DevDash-Setup-0.2.0.exe` (~81 MB)

## Notes

- Screenshots use Electron's own offscreen BrowserWindow, not puppeteer, to keep installer size down.
- Sentry integration is optional: without `sentryDsn` + `sentryAuthToken`, error budget falls back to scanning each project's `logs/` folder for `ERROR` lines.
