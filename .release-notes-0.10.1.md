# DevDash v0.10.1

Hover tooltips on every project action button.

## New

**Tooltips on every button.** Hover over any action button on a project card and a native tooltip shows exactly what the button does, including context like the live URL, GitHub URL, or dev script it will run.

Covered buttons: Run / Stop, Logs, Env, Folder, VS Code, GitHub, Live, Pull, Commit, Diff, PRs, Redeploy, Deploy, Release.

Disabled buttons now explain why (e.g. "No GitHub URL configured", "No live URL set — deploy first or add it in project settings", "Git not available for this project").
