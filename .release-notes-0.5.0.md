DevDash 0.5.0 is a polish release focused on chat quality and theming.

### Added
- **Markdown rendering in chat** — assistant responses now render with full markdown support, including headings, lists, links, tables, blockquotes, inline code, and fenced code blocks. Rendering uses `marked` with `DOMPurify` sanitization. Every fenced code block includes a Copy button, and external links open in the default browser.
- **Keyboard shortcuts overlay** — press `?` anywhere outside an input to open a categorized list of every shortcut. Press Esc to close.
- **Light mode** — Settings exposes a three-way theme selector: Dark, Light, or System. System tracks the operating system color scheme and updates live when it changes.

### Changed
- The `Appearance & startup` section replaces the former placeholder Dark-mode toggle with the new theme selector.
- User messages in chat remain plain text; only assistant and system messages are rendered through the markdown path.

### Known issues
- Syntax highlighting inside code fences is not yet implemented; code blocks render with a monospace font but without language-based coloring.
- A handful of deep panel-tint variations may still appear dark in niche surfaces under Light mode and will be tuned in a follow-up.
- Remaining roadmap items (encrypted configuration export and import, scheduled automations, pull request status, diff viewer, database health, Render service metrics, Vercel analytics, project detail breadcrumb, and the Electron major-version upgrade) remain deferred.

### Install
Download `DevDash-Setup-0.5.0.exe` below. Please uninstall the previous version first for a clean install. Existing configuration and project data are preserved across upgrades.
