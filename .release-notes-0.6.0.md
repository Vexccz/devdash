DevDash 0.6.0 adds syntax highlighting to chat and encrypted backup for the full configuration.

### Added
- **Syntax highlighting in chat** — code fences in assistant messages now render with the `highlight.js` GitHub Dark theme. Supports 190+ languages via the common language bundle, including JavaScript, TypeScript, Python, Bash, JSON, YAML, Markdown, HTML, CSS, Go, and Rust.
- **Encrypted configuration backup** — a new Backup & restore section in Settings exports the full configuration (projects, API tokens, preferences) to an encrypted file protected by a passphrase, and re-imports on another machine. Encryption uses AES-256-GCM with scrypt key derivation; the same passphrase is required to decrypt.

### Changed
- Chat code blocks now render with syntax-aware coloring. Plain text blocks and user messages remain unchanged.

### Known issues
- Importing a configuration does not refresh the running process; Settings and project views reload on the next app start.
- Pull request status, scheduled automations, diff viewer, database health, Render service metrics, Vercel analytics, and project detail breadcrumb navigation remain deferred.

### Install
Download `DevDash-Setup-0.6.0.exe` below. Please uninstall the previous version first for a clean install. Existing configuration is preserved across upgrades.
