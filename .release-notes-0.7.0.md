DevDash 0.7.0 brings git diff and GitHub pull request status into the main project view.

### Added
- **Diff viewer** — a new Diff action on each project card opens a modal showing the project's git diff, toggleable between Unstaged and Staged views. Hunks are color-coded (additions green, removals red, hunk headers indigo) for quick review.
- **Pull request status** — a new PRs action (enabled when a GitHub URL is configured) lists up to 20 open pull requests using the GitHub CLI. Each entry shows number, title, author, head branch, relative update time, draft status, and mergeability. Clicking a row opens the pull request in the default browser.

### Requirements
- The PR status feature requires the GitHub CLI (`gh`) to be installed and authenticated. If it is missing, the modal surfaces a clear error with the `gh auth login` hint.

### Known issues
- Pull request listings are scoped to the upstream repository detected from the local clone. Forks with a different origin may require manual repository selection in a follow-up release.
- Scheduled automations, database health checks, Render service metrics, Vercel analytics, project detail breadcrumb, and the Electron major-version upgrade remain deferred.

### Install
Download `DevDash-Setup-0.7.0.exe` below. Please uninstall the previous version first for a clean install. Existing configuration is preserved across upgrades.
