# DevDash v0.9.1

Onboarding wizard release. First-run experience replaces the old empty dashboard.

## New

**Onboarding wizard.** First launch now walks through four quick steps:

1. **Welcome** — what DevDash is for, no input required
2. **API tokens** — Vercel and Render token fields with inline test buttons, skippable
3. **Smart scan** — pick a parent folder, DevDash scans and auto-selects repos with git remotes or matching deploys
4. **Preferences** — theme (dark/light/system), auto-launch toggle, poll interval slider

Each step has a Skip option. A dot indicator shows progress. The wizard closes and opens the dashboard when done.

**Re-run wizard anytime.** New button in Settings → About: "Re-run setup wizard". Useful after adding a new Vercel/Render token to re-scan for matches, or when cloning a fresh batch of projects.

## How first-run detection works

A new `onboardingComplete` flag is stored in `config.json`. The wizard triggers when the flag is missing or false. Skipping the whole wizard sets the flag so it doesn't keep appearing.

## Install

Download `DevDash-Setup-0.9.1.exe` below. Existing users with a populated config will not see the wizard unless they click "Re-run setup wizard" in Settings.
