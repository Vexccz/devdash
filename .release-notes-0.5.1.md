Adds Ollama cloud support so Chat works without a locally running Ollama server.

### Added
- A dedicated **Ollama chat** section in Settings with an editable **Base URL** field and a password-masked **API key** field.
- When an API key is configured, the Chat tab attaches an `Authorization: Bearer <key>` header to every request to `/api/tags` and `/api/chat`. Leave the key blank for a local Ollama server.

### Fixed
- Chat failed to load models when only an Ollama cloud endpoint was available because requests were made without an Authorization header.

### Usage
- For local: set Base URL to `http://localhost:11434` and leave the API key blank.
- For Ollama cloud: set Base URL to `https://ollama.com` and paste the key from `ollama.com/settings/keys`.

### Install
Download `DevDash-Setup-0.5.1.exe` below. Please uninstall the previous version first for a clean install. Existing configuration is preserved across upgrades.
