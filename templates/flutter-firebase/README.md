# {{DISPLAY_NAME}}

Flutter mobile app with Firebase (Auth, Firestore, Storage)

## Getting Started

```bash
# Install Flutter dependencies
flutter pub get

# Set up Firebase
# 1. Create a Firebase project at https://console.firebase.google.com
# 2. Run: flutterfire configure
# 3. Copy google-services.json (Android) and GoogleService-Info.plist (iOS)

# Run the app
flutter run
```

## Stack

- **Framework:** Flutter 3.x
- **Auth:** Firebase Authentication (Email/Password + Google)
- **Database:** Cloud Firestore
- **Storage:** Firebase Storage
- **State:** Provider

## Project Structure

```
lib/
├── main.dart           # App entry point
├── firebase_options.dart # Generated by FlutterFire CLI
├── models/             # Data models
├── screens/            # UI screens
├── services/           # Firebase service wrappers
└── widgets/            # Reusable widgets
```

## Environment Variables

See `.env.example` for configuration. Firebase config is handled via `firebase_options.dart`.

## Build

```bash
# Android APK
flutter build apk --release

# iOS
flutter build ios --release
```
